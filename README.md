                  ## Recent Product Thinking

                  ### Problem-Solving > Code
                  Built CommuniEats spec entirely with Claude Code as my "product partner." The hard part wasn't shipping the app—it was **articulating the trust problem clearly enough to solve it**. Took 3 iterations to frame it properly.

                  **Lesson:** Most product challenges aren't execution problems; they're framing problems.

                  ### Market Validation
                  JoyRun tried the neighbor-delivery model in 2017, raised $10M, and was acquired by Walmart in 2020. Still not mainstream. Why?
                  - ✅ The technology works
                  - - ✅ The unit economics are viable
                    - - ❌ The core problem (trust/density/timing) remains unsolved
                     
                      - **Takeaway:** Building a feature is easy. Solving the underlying problem is hard.
